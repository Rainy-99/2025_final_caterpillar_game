# 🐛 Caterpillar Game（毛毛蟲遊戲）

這是一個使用 **Python + Tkinter** 製作的單人毛毛蟲（貪食蛇）遊戲，  
目前已完成基礎遊戲機制與多葉子系統，並預留結構以擴充為拼字遊戲。

## 🎮 目前已完成的功能

### 基本玩法

- 單人毛毛蟲（貪食蛇）遊戲
- 使用格子座標系（GRID_W × GRID_H）
- 可使用方向鍵 **Up / Down / Left / Right** 操作
- 撞牆或撞到自己會 **Game Over**
- 按 **R** 可重新開始遊戲
- 即時顯示分數

---

### 🍃 多葉子系統（已完成）

- 遊戲中 **可同時存在多顆葉子**
- 目前設定：
  - `LEAF_COUNT = 8`
- 遊戲規則：
  - 吃到任一顆葉子：
    - 毛毛蟲身體變長
    - 分數 +1
    - 被吃掉的葉子會移除
    - 系統會自動補上一顆新葉子
- 葉子生成規則：
  - 不會生成在毛毛蟲身體上
  - 不會與其他葉子重疊

---

### 🎨 畫面顯示

- 視窗大小依格子數與每格像素大小自動調整
- Canvas 背景色已設定
- 畫面包含：
  - 毛毛蟲（頭與身體顏色不同）
  - 多顆葉子
  - 分數顯示
  - Game Over 提示文字

---

## ⚙️ 遊戲設定（config.py）

可在 `game/config.py` 中調整：

- 地圖大小（`GRID_W`, `GRID_H`）
- 每格像素大小（`CELL`）
- 更新速度（`TICK_MS`）
- 葉子數量（`LEAF_COUNT`）
- 顏色設定
